Grid System

Per grid system si intende un insieme di regole CSS che vanno a comporre un sistema per costruire layout composto principalmente da 3 elementi:


  • Container

    Spesso identificato con la classe .container, rappresenta il contenitore che andrà a raggruppare al suo interno la griglia.


  • Row

    Spesso identificato con la classe .row, rappresenta la riga che ha la funzione di contenitore per le varie colonne.

  • Column

    Spesso identificato con la classe .col, rappresenta l'ultima parte del grid system, ovvero la colonna che racchiude il contenuto.
