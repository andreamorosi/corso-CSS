Cosa è il CSS

Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML or XML (including XML dialects such as SVG or XHTML). CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

Prendiamo per esempio il seguente codice:

p { color: blue; }

Questo è un CSS ruleset, ovvero una regola CSS.
Una regola CSS è formata così:

  • Selector, ovvero il nostro p.
  • Property, ovvero il nostro color.
  • Property value, ovvero il nostro blue.

L'insieme di property e property value forma una Declaration.

  • {} : le parentesi graffe mi servono per contenere la regola CSS.
  • : : i due punti mi servono per separare property dal property value.
  • ; : il punto & virgola mi serve per separare una declaration dall'altra.
