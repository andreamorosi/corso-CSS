Come applicare il CSS

Ci sono 3 modi per applicare del CSS ad un documento HTML:

  • Inline
  • Internal
  • External

Inline

Viene scritto direttamente nel documento HTML, precisamente all'interno dell'elemento, nel attributo style.

Hello World
