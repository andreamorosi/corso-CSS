Box Model

Ogni elemento HTML viene rappresentato come un box rettangolare, caratterizzato da 4 parti:

  • Content
  • Padding
  • Border
  • Margin

Content

E' la parte dove viene rappresentato il contenuto dell'elemento, tipo del testo, delle immagini o altro.

Padding

Rappresenta un'area trasparente attorno al contenuto.
Quest'area si gestisce tramite padding-top, padding-right, padding-bottom e padding-left, oppure tramite lo shorthand padding, che hanno come valore una grandezza, ma non può essere un valore negativo.

CSS
                    
        .elemento {
            padding: 10px 20px 30px 40px;
        }

Border

Il bordo è l'area che viene subito dopo il padding e, a differenza di quest'ultima, può essere modificata di aspetto e conferisce un dettaglio grafico aggiuntivo all'elemento.
E' molto utile lo shorthand border per lavorare sul bordo, che raggrupa il border-width border-style e border-color

CSS
                    
        .bordo {
            border: 2px solid red;
        }

Margin

Il margine rappresenta l'ultima area del box model, è trasparente come il padding ma quando viene a contatto insieme al margine di un altro elemento (vale solo per i margini verticali) si uniscono e il risultato viene dato dal margine più grande: questo fenomeno si chiama margin collapsing e ci sono eccezioni in cui non si verifica, tipo tra elementi posizionati in maniera assoluta.
Per quanto riguarda la proprietà, è uguale al padding, ovvero margin-top, margin-right, margin-bottom e margin-left, oppure tramite lo shorthand margin, e il valore è una grandezza ma può avere valori negativi.

CSS
                    
        .elemento {
            margin: 5px 10px 15px 5px;
        }

Box sizing

Il box-sizing è una proprietà CSS che stabilisce il modo in cui viene calcolata la grandezza di un elemento, ovvero della width e della height.
Questa proprietà può avere 2 valori (oltre i 2 valori di base, initial o inherit):

  • content-box, ovvero la width e l'height dell'elemento includono solo il contenuto.
  • border-box, ovvero la width e l'height dell'elemento includono il contenuto, il padding e il border.
